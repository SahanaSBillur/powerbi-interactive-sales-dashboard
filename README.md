# powerbi-interactive-sales-dashboard
Project Title: Interactive Sales Analysis Dashboard using Power BI
Headline: A data-driven Power BI dashboard to monitor sales performance, identify trends, and generate actionable business insights.

Short Description:
This mini project showcases an interactive Sales Analysis Dashboard developed in Power BI.
It provides a comprehensive view of key metrics such as total sales, profit, and quantity, along with category-wise and time-based analysis.
The dashboard helps stakeholders quickly understand performance patterns, compare segments, and make informed decisions.

echnologies Used:
Power BI Desktop
DAX (for calculated measures & KPIs)
Power Query (for data transformation)

Features / Highlights:
KPI Cards – Total Sales, Total Profit, Total Quantity
Time-based analysis (Monthly/Yearly trends)
Category and Sub-category performance breakdown
Top-performing products visualization
